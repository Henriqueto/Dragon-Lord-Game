# Dragon-Lord-Game
Um RPG de turnos desenvolvido inteiramente em Java como projeto acadêmico, focado na implementação de mecânicas clássicas de combate estratégico. O jogo explora conceitos fundamentais de programação orientada a objetos, como encapsulamento, reutilização de código e organização modular.

## Funcionalidades

- Sistema de combate por turnos
- Personagens com atributos (vida, ataque, defesa, etc.)
- Inimigos com comportamentos distintos
- Sistema de escolhas estratégicas a cada turno

## Tecnologias

- Java
- Programação Orientada a Objetos (POO)

## ▶️ Como executar

1. Certifique-se de ter o Java instalado (Java 8 ou superior)

2. Baixe o arquivo `.jar` disponível no repositório

3. Execute o jogo com o comando:

```bash
java -jar DragonLord.jar
```
## Aprendizados

- Aplicação prática de POO
- Organização de código em Java
- Lógica de sistemas de combate por turnos
